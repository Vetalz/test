1. Перейдите в папку html и выведите список всех папок (без файлов)
	cd var/www/html            # переход в папку html
	ls -d */                   # вывести список всех папок (без файлов)

2. Объедините все файлы с расширением .txt в один /all.txt
	cat $(find ./var -name '*.txt') > all.txt     # с помощью команды find получаю все файлы *.txt; с помощью cat объединяю их содержимое в один файл

3. Удалите пустые строки из файла error.txt
	cd var/www/html/logs                             # перейти в папку с файлом
	mv error.txt error_old.txt                       # переименовую файл
	grep -v '^$' error_old.txt > error.txt           # с помощью grep удаляю пустые строки и полученный результат перенаправляю в новый файл
	rm error_old.txt                                 # удаляю старый файл
	
4. Выведите список файлов с расширением .jpg в папке images
	cd var/www/html/images                           # перейти в папку с картинками
	find ./ -name '*.jpg'                            # показать список файлов
	
5. Удалите все содержимое в файле access.txt не удаляя сам файл
	cd var/www/html/logs                             # перейти в папку с файлом
	> access.txt                                     # удалить все содержимое файла

6. Выведите список файлов в папке images размер которых более 2 мегабайт
	cd var/www/html/images                           # перейти в папку с картинками
	find -size +2048                                 # показать файлы размером больше 2мб
